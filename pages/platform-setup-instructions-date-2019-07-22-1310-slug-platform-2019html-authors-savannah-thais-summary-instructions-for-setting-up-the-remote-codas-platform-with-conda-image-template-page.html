Remote CoDaS Platform Setup

First, login to the platform using Globus with your CERN or university credntials.

  1. Using the bar at the top of the screen select Services -> “Private JupyterLab”

Then select "New Personal Lab"

  1. Fill out the form with appropriate settings to configure your lab
  2. Name: this needs to be unique for each instance so use something with e.g. your actual name
  3. Password: set your own password
  4. Time to live: 5 days is fine
  5. GPUs: 1 is fine
  6. CPUs: 2
  7. Memory: leave at 8
  8. Check out your github repo: https://github.com/Atcold/pytorch-Deep-Learning-Minicourse.git
  9. Select preferred image: this must be set to the conda image

Click start. It will likely take a few minutes to setup; when it is running it will appear in your list of labs

  1. Launch the lab by clicking on the link and entering your password from the setup step. You know have a CoDaS-HEP environment when many useful packages already installed. You can add additional packages to this environment or create an entirely new one if desired.
